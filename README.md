# ExaTrack

This repository contains the ExaTrack project which makes the syntesis between our previous tools ExTrack and aTrack by designing an userfriendly tracking analysis tool able to account for multiple states of various types of motions and complex transitions between them. More precisely, this tool enable to account for immobile, diffusive, directed and confined motion and for transitions between these types of motion. This tools enable to characterize the different types of motion and the transition kinetics of their samples.
